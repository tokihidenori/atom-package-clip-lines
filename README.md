# clip-lines package

Clipping file lines and line numbers

# How to use

1. Selected File Lines.
2. Continue `clip-lines:copy` until you get entry you want.

# clip sample

```
/Users/.../github/copy-file-lines/README.md: 8

2. Continue `clip-lines:copy` until you get entry you want.
```

# Keymap

- Mac: ctrl + opt + c
- Win: ctrl + alt + c

# Config

- lineEndingCode

  default: \n

- lineNumberSepalater

  default: |

- enableMarkDownFormat

  default: true
